ABOUT THIS WEB APPLICATION

Welcome to Amanda's online shoe catalogue.
This web applictaion anables people to search and buy stock,
also enables the admin to add new stok and update the existing one.
If the stock is sold out then that stock should dissapear from the database.

As the customer you are able to:

  • See tha available shoes in the shop
  • Search the shoes they want using the text area
  • Search the shoes they want using the dropdowns
  • buy the shoes that they want

As the admin i am able to:

  • Add new stock
  • Update the existing stock
